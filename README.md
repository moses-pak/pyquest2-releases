# PyQuest 2 🤖🐍

**Learn to code with an AI partner** — the sequel to
[PyQuest](https://github.com/moses-pak/pyquest-releases), built for analysts
who finished Phase One. Prompting that works on the first try, catching the
AI's mistakes before they reach a report, AI-assisted debugging, IDE
assistants, agentic coding, and brainstorming real projects with AI — capped
by a project you build with your own AI tool.

Same game, new quest: short lessons, live Python in your browser,
auto-checked exercises, XP and streaks.

## Before you start

Finish **PyQuest (Phase One)** first — this course assumes you can read and
write Python and know basic pandas. New here? Start at the
[PyQuest releases page](https://github.com/moses-pak/pyquest-releases).

## Get started (2 minutes)

1. **[Download PyQuest2.zip](../../releases/latest/download/PyQuest2.zip)**
2. Right-click the zip → **Extract All…** and extract it anywhere you like
   (Desktop or Documents is fine).
3. Double-click the extracted `PyQuest2.exe`. If Windows still shows
   *"Windows protected your PC"*, click **More info → Run anyway** (the
   warning appears because the file isn't from a big-name publisher — not
   because anything is wrong).
4. A small console window opens, then your browser opens the quest map.
   Keep the console window open while you learn; close it when you're done.

Prefer the bare exe? It's on the [releases page](../../releases/latest)
too — the zip just gets past SmartScreen download blocks more smoothly.

That's it. No installs, no setup — you need a browser and an internet
connection. Your progress saves automatically and picks up where you left
off. PyQuest and PyQuest 2 run side by side without touching each other's
progress.

## Updating to a new version

Download the newest `PyQuest2.zip` from the
[releases page](../../releases/latest), extract it, and replace your old
`PyQuest2.exe` with the new one.
**Your progress is kept automatically** — it lives in your browser, not in
the app file.

Tip: the **Export progress** button on the quest map saves a backup file.
**Import progress** restores it — even on a different browser or computer.

## One rule worth repeating

The course covers this in Chapter 0, but it can't be said too often:
**never paste real patient data or credentials into an AI tool.** Everything
in this course runs on made-up data, and everything you build from it
should too.
